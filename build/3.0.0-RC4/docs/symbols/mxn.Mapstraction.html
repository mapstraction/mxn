Class Detail

mxn.Mapstraction(element, api, debug)

Creates and loads a Mapstraction map into a specified HTML element. The following mapping APIs are supported by Mapstraction:

  • cloudmade - CloudMade
  • esri - ESRI ArcGIS
  • google - Google v2
  • googlev3 - Google v3
  • leaflet - Leaflet
  • mapquest - MapQuest
  • microsoft - Microsoft Bing v6
  • microsoft7 - Microsoft Bing v7
  • nokia - Nokia Here
  • openlayers - OpenLayers
  • openmq - MapQuest Open
  • openspace - Ordnance Survey OpenSpace
  • ovi - Nokia Ovi
  • yahoo - Yahoo (obsoleted)
  • yandex - Yandex
Parameters:
{string} element
The HTML element to replace with a map.
{string} api
The API ID of the mapping API to use; if omitted, the first loaded provider implementation is used.
{boolean} debug Optional
optional parameter to turn on debug support; this uses alert panels for unsupported actions.

  • addControls(args)

    addControls adds (or removes) controls to/from the map. You specify which controls to add in the object literal that is the only argument.

    To remove all controls from the map, call addControls with an empty object literal as the argument.

    Each time addControls is called, those controls present in the args object literal will be added; those that are not specified or as specified as false will be removed.

    args = {
	 pan:		true,
	 zoom:		'large' | 'small',
	 overview:	true,
	 scale:		true,
	 map_type:	true,
}

  • {Object} addTileLayer(tile_url, opacity, label, attribution, min_zoom, max_zoom, map_type, subdomains)

    Adds a Tile Layer to the map.

    Requires providing a templated tile URL. Use {S}, {Z}, {X}, and {Y} to specify where the parameters should go in the URL. {S} is the (optional) subdomain to be used in the URL. {Z} is the zoom level. {X} and {Y} are the longitude and latitude of the tile.

    Sample templated tile URLs are :-

    • OpenStreetMap - http://{S}.tile.openstreetmap.org/{Z}/{X}/{Y}.png
    • Stamen Toner - http://tile.stamen.com/toner/{Z}/{X}/{Y}.png
    • MapQuest OSM - http://otile{s}.mqcdn.com/tiles/1.0.0/map/{z}/{x}/{y}.jpg
    • MapQuest Open Aerial - http://otile{s}.mqcdn.com/tiles/1.0.0/sat/{z}/{x}/{y}.jpg
    Parameters:
    {string} tile_url
    Template URL of the tiles.
    {number} opacity
    Opacity of the tile layer - 0 is transparent, 1 is opaque. (default=0.6)
    {string} label
    The label to be used for the tile layer in the Map Type control
    {string} attribution
    The attribution and/or copyright text to use for the tile layer
    {Int} min_zoom
    Minimum (furthest out) zoom level that tiles are available (default=1)
    {Int} max_zoom
    Maximum (closest in) zoom level that the tiles are available (default=18)
    {boolean} map_type
    Should the tile layer be a selectable map type in the layers palette (default=false)
    {String|Array} subdomains
    List of subdomains that the tile server in tile_url refers to. Can be specified as a string "abc" or as an array [1, 2, 3]
    Returns:
    {Object} The tile layer object

  • runDeferred()
    Run any queued provider API calls for the methods defined in the provider's implementation. For example, if defferable in mxn.[provider].core.js is set to {getCenter: true, setCenter: true} then any calls to map.setCenter or map.getCenter will be queued up in this.onload. When the provider's implementation loads the map, it calls this.runDeferred and any queued calls will be run.
