Class Detail

mxn.Radius(center, quality)
Creates a Mapstraction Radius for drawing circles around a given point. Note that creating a radius performs a lot of initial calculation which can lead to increased page load times.
Parameters:
{mxn.LatLonPoint} center
Central mxn.LatLonPoint of the radius
{number} quality
Number of points that comprise the approximated circle (20 is a good starting point)
